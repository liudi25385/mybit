## 声明：本项目仅用于个人演练，不可用于商用
### 项目名称： mytik
基于ARKTS语言开发，鸿蒙next版本
***
### 功能介绍：
* 短视频播放、暂停、滑动
* 登录模块
* 异步网络请求，接口
* 评论、点赞
* 新闻类展示
* 类似Android的CoordinateLayout效果，下拉图片跟随缩放
* 过年抽生肖卡活动
* zhi护宝首页
* 瀑布流展示
* Ability使用，携参
* ...
